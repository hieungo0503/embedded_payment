Course Modules

  • Module 1: Intro

    Gain a foundational understanding of the Internet of Things (IoT), its applications, and the course structure. Explore the basics of IoT devices and their role in real-world scenarios, setting the stage for hands-on development.

  • Module 2: Zephyr and Terraform

    Master the Zephyr RTOS for embedded development and use Terraform to automate cloud infrastructure setup. Learn how to integrate these tools to create a scalable IoT environment for your device.

  • Module 3: Cellular Networks

    Understand LTE infrastructure and cellular network fundamentals essential for IoT connectivity. Explore how cellular networks enable reliable communication for IoT devices.

  • Module 4: Network and Transport (TCP/IP)

    Dive into the TCP/IP stack, learning how to implement network and transport layers for IoT devices. Gain skills to ensure robust data transmission in embedded systems.

  • Module 5: Application Protocols

    Explore application protocol layers, including MQTT and CoAP, to enable efficient communication between IoT devices and cloud services. Learn to design protocol-based solutions.

  • Module 6: Power and APIs

    Learn power management techniques for IoT devices and develop RESTful APIs to connect devices with cloud platforms. Master energy-efficient design and API integration.

  • Module 7: Application Design

    Design IoT applications from scratch, focusing on user requirements and system architecture. Apply embedded engineering skills to create functional software for your device.

  • Module 8: Firmware Updates

    Implement firmware update mechanisms for IoT devices, ensuring seamless upgrades in the field. Understand the process of maintaining and updating device software.

  • Module 9: OTA Updates and Security

    Master over-the-air (OTA) updates and implement security measures to protect IoT devices. Learn encryption, authentication, and secure update deployment techniques.

  • Module 10: Testing and CI

    Develop testing strategies and continuous integration (CI) pipelines for IoT products. Gain hands-on experience in validating device functionality and automating builds.

  • Module 11: Monitoring and Diagnostics

    Implement monitoring and diagnostic tools to track IoT device performance and troubleshoot issues. Learn to analyze data and ensure long-term product reliability.

  • Slides and Assignments

    Comprehensive Lecture Materials and Practice Tasks with Step-by-Step Guidance.
